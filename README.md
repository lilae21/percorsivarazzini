# Percorsi varazzini
Project work per il master di II livello in Digital Humanities (Università degli Studi di Milano), a.a. 2024-2025.

Il sito *Percorsi varazzini* è un prototipo di portale web divulgativo nell'ambito di una valutazione di possibili progetti che potrebbero essere intrapresi per la valorizzazione e la divulgazione, riscoprendo la storia industriale e turistica della Varazze di inizio Novecento: una città perduta, dal punto di vista urbanistico, a seguito delle diverse trasformazioni avvenute soprattutto negli ultimi trent'anni.
