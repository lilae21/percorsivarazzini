## 1. Informazioni Generali

Questo sito web è un prototipo hostato su GitHub Pages. Il titolare del sito non raccoglie, memorizza o processa alcun dato personale degli utenti in modo diretto.

## 2. Dati Raccolti

### 2.1 Dati raccolti direttamente dal sito
Questo sito **non raccoglie** né memorizza alcun dato personale degli utenti. Non vengono utilizzati cookie proprietari o sistemi di tracciamento.

### 2.2 Servizi di terze parti

#### GitHub Pages
Il sito è hostato su GitHub Pages. GitHub potrebbe raccogliere dati tecnici di base come indirizzi IP per finalità di sicurezza e performance. Per maggiori informazioni, consulta la [Privacy Policy di GitHub](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement).

#### Google Fonts
Questo sito utilizza Google Fonts per il caricamento dei caratteri tipografici. Quando visiti il sito, il tuo browser effettua una richiesta ai server di Google per scaricare i font necessari. Durante questa operazione, Google può raccogliere:
- Indirizzo IP
- Informazioni tecniche sul browser e sul dispositivo utilizzato
- Data e ora della richiesta

Questi dati sono raccolti e processati da Google secondo la propria privacy policy. Per maggiori informazioni: [Google Privacy Policy](https://policies.google.com/privacy).

## 3. Cookie

Questo sito **non utilizza cookie** proprietari. 

I servizi di terze parti menzionati (GitHub Pages, Google Fonts) potrebbero utilizzare cookie tecnici o memorizzare dati temporanei per il corretto funzionamento del servizio.

## 4. Base Giuridica del Trattamento

Non essendoci raccolta diretta di dati personali da parte del titolare del sito, non si applica alcuna base giuridica per il trattamento. I servizi di terze parti operano secondo le proprie policy e basi giuridiche.

## 5. Diritti degli Utenti

Poiché questo sito non raccoglie dati personali, non è necessario esercitare diritti di accesso, rettifica o cancellazione verso il titolare del sito. Per questioni relative ai dati raccolti dai servizi di terze parti, è necessario contattare direttamente GitHub o Google.
